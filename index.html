My Profile

Halo nama saya Wahyu Dwi Prakoso. Saya adalah mahasiswa Teknik Informatika yang gemar mendesain dan membuat algoritma. Saya suka membuat WPAP( Wedha's Pop Art Potrait ) dan desain abstrak. Saya juga telah berhasil membuat aplikasi - aplikasi seperti web ecomers, web kasir restoran, dan web sistem informasi kkn.

Saat Masuk waktu SMK, saya mengambil Jurusan RPL (Rekayasa Perangkat Lunak). Jurusan ini merupakan studi ilmu yang saya minati hingga sekarang. Disinilah saya menemukan teman serta passion mengkoding. Di SMK juga saya pertama kali melakukan praktik kerja. Waktu itu saya praktek di toko ELcom milik salah satu guru smk saya. Toko Elcom merupakan tempat yang menyediakan layanan berupa servis dan instal ulang laptop, print dan fotocopy, juga jual beli alat sekolah. Disana saya bekerja sebagai pegawai yang bisa diandalkan. Hingga saya berhasil melanjutkan pendidikan saya hingga ke perguruan tinggi.

Saat ini saya sedang menempuh jurusan Teknik Informatika. Saya paling menggemari mata kuliah pemrogaman web dan basis data. saya lebih familiar dengan php dan mysql dibandingkan bahasa pemrogaman lainnya.
